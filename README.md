# Angular Shopping App

[__Live Demo__](https://ng-shop-app.herokuapp.com/)

```bash
  
$ npm i -g gulp nodemon && npm install $$ bower install
  
```

now run Gulp
```bash
  
$ gulp
  
```

and in a second terminal run Nodemon. check out [port 3000](http://localhost:3000/)
```bash
  
$ nodemon
  
```
